Principais formatações

Negrito ou destaque

nesta fraser temos um termo em destaque ou Negrito usando a tag B (não sermantica).

nesta frase temos um termo em destaque com a tag strong (semantica)

Italico

Nesta frase temos um termo em itálico usando a tag I (não semantica)

Nesta frase temos um termo em ênfase com a tag em (semantica)

Texto marcado

Podemos criar um texto marcado para destacar e dar ênfase em uma frase.

Também é possivel marcar textos com ouureas cores usando style dentro da tag ou do documento (head)

Texto grande e pequeno

Para o texto grande é possível utililzar a tab BIG (não semantica)

para o texto pequeno é possivel usar a tag small

Texto deletado

Quando for necessario colocar uma informação que não deve ser considerada pode ser usada a tag del.

Texto inserido

Para sublinhar uma frase, podemos usar a tag ins ou então a tag U

Textos sobreescritos

Podemos escrever numeros com notações cientificas utilizando a tag sup. 10-9x1

Textos subscritos

Podemos usar a tag sub para escrever fórmulas quiímicas. H2O
