Trabalhando com listas

Lista Ordenada

Para fazer uma lista com uma contagem de itens, é possível usar a tag ol, que possui complementos li, que representam os itens a lista.

  1. Primeiro
  2. segundo
  3. terceiro
    4. ...
é possivel também personalizar a lista com o atributo type
  1. quarto
  2. quinto
  3. sexto

Lista não ordenada

Segue o mesmo formato da OL, porém com a tag ul. este tipo de lista não possui qualquer tipo de contagem

  • salada
  • bolo de milho
  • sorvete
